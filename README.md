# riic-rotation

Generate 12 hour rotations for your Arknights base. [Hosted on Vercel.](https://riic-rotation.vercel.app/)


## Usage
1. Select operators you own and their elite levels.
2. Add facilities to reflect your base layout.
3. Click "Schedule".
